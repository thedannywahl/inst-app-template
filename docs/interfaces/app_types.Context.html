      • (): void

      • Returns void

  • isOpen: boolean

    Indicates whether the App Modal is currently open.

  • props: ModalProps

    Represents the props of the App Modal Components.

    Remarks

    the props object maps the following keys to their respective components:

    • modal to the Modal component
    • header to the Modal.Header component
    • body to the Modal.Body component
    • footer to the Modal.Footer component Modal.label and Modal.children are omitted from the props object requirements. label can be optionally included in the modal object.

    Example

    {
         *   modal: { size: "medium" },
         *   header: { spacing: "compact" }
         * }
